# Projeto Anhembi

Aplicativo de voluntariado — site criado a partir do design no Figma.

O app conecta voluntários a causas e ONGs, com acompanhamento de horas de
voluntariado, notificações e avaliação das causas.

## Telas

| Arquivo | Tela |
|---|---|
| `index.html` | Login |
| `home.html` | Home com os cartões das causas |
| `feedback.html` | Lista de causas para avaliar |
| `avaliar.html` | Formulário "Deixe seu feedback!" com estrelas |
| `perfil.html` | Perfil do usuário |
| `horas.html` | Acompanhamento das horas de voluntariado |
| `notificacoes.html` | Lista de notificações |
| `notificacao.html` | Detalhe da notificação |
| `filtragem.html` | Filtros de busca (causas, ONGs, habilidades, vagas) |

## Como abrir

Basta abrir o arquivo `index.html` no navegador — não precisa instalar nada.

## Tecnologias

- HTML5
- CSS3
- JavaScript

Design feito no Figma e convertido em código com a ajuda do Claude.
